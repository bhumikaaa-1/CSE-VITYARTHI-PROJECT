 EasyBook - Movie Ticket Booking System

A simple, clean and fully working movie ticket booking web application.

 Features
- User Registration (Name + Password)
- Secure Login and Logout
- View list of available movies
- Choose any movie and book tickets
- Select number of tickets (1 to 10)
- View all your bookings
- Cancel any booked ticket with one click
- Fully responsive – works on mobile and desktop

 
 How to Run the Project

bash
 Clone the repository
git clone https://github.com/YOUR_USERNAME/easybook-movie-ticket-booking.git

 Go to project folder
cd easybook-movie-ticket-booking

 If it's pure HTML/CSS/JS → just open index.html in browser

 If you have a backend (Node.js / PHP / Python):
 Check the backend folder and follow its instructions
 Example for Node.js:
cd backend
npm install
node server.js
